Succubus Club

{% trans trimmed succubus_club=external("https://succubus-club.net", "Succubus Club") %} {{ succubus_club }} is a new online play platform actively developed by a member of the VTES community. There are no accounts nor tracking. All data stays in the browser's local storage. Multiplayer supports two communication modes: {% endtrans %}

    {% trans trimmed %}
  • Direct mode: Players connect directly to each other, so there is no authoritative server watching over the game. Faster, but cheating is possible, and players are expected to rely on trust and good sportsmanship.
  • Succubus Club Server (SCS): Slower, but an authoritative server handles the game state and prevents players from taking a look at hidden cards they are not supposed to see.
    • {% endtrans %}

{% trans trimmed %} This web application has been developed as a modern replacement for LackeyCCG. Keyboard shortcuts can be configured in the preferences, and decks can be loaded from any of the large deckbuilding websites. {% endtrans %}

{% trans trimmed %} There's a log describing all actions and a card zoom area like in LackeyCCG. The interface uses keyboard shortcuts, left-click default actions and radial menus, as well as right-click context menus, to interact with the cards. {% endtrans %}

{% trans trimmed %} Compared to LackeyCCG, some practical new features have been added: {% endtrans %}

    {% trans trimmed %}
  • An influence shortcut on uncontrolled cards, to add blood and remove pool in a single move.
  • Guided action resolution with impulse materialization and stealth/intercept counters.
  • The ability to declare targets that show in the log.
  • Automatic card attachment, to stack cards on one another.
    • {% endtrans %}

{% trans trimmed %} Double click on a card to lock or unlock it, click on the decks (library or crypt) to draw, click on the Edge icon to take or release it, and use drag & drop to move cards around. {% endtrans %}
