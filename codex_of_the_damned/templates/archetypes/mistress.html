Highlights

This is a bleed & vote deck starring {{ mistress_fanchon }}. Her ability lets her fetch an {{ alastor }} to equip with an {{ helicopter }}, paying only 1 pool.

{{ villein }}, {{ zillah_s_valley }} and {{ information_highway }} are used to get multiple big capacity vampires into play.

This version uses only the {{ DOM }}{{ OBF }} disciplines, a dreadful combination. {{ obedience }} and {{ mental_maze }} are used for combat defence.

Tips & Tricks

The key idea of this archetype is to abuse {{ mistress_fanchon }}'s ability to fetch the right card at the right time. The focus of the deck is to make sure she can do that and bleed every turn.

{{ scourge_of_the_enochians }} is included to cripple breed constructs, as they can be overwhelming for this slower deck.

Variants

The {{ alastor }} + {{ helicopter }} combination is not mandatory. Other methods can be found to improve {{ a_b }}, for example using {{ flurry_of_action }} like in {{ benoît_moyen_s_deck }} from 2010.

Danilo Torrisi played a {{ nice_combination }} in 2018: The {{ might_of_the_camarilla }} alongside {{ possession }} to get more Inner Circle into play fast and cheap.
