Highlights

This is a stealth & bleed starring {{ thucimia }}, helped by a couple of lower capacity sidekicks.

The idea is to influence {{ thucimia }}, and bleed while also continuously stacking more permanent bleed enhancers on her thanks to a second action from {{ freak_drive }}, {{ dabbler }} and {{ forced_march }}.

Bleed power is ludicrous. {{ dem }} or {{ qui }} are used for bleeding with {{ kindred_spirits }} and {{ loss }}, and stealth is provided mostly by Obfuscate {{ OBF }} modifiers, but also by {{ confusion }}.

Between her ability and {{ learjet }}, {{ thucimia }} allows for efficient cycling, so she can equip with bleed enhancers like {{ dive_into_madness }}, {{ j_s_simmons }}, {{ tasha_morgan }}, and most importantly {{ enkil_cog }}. The {{ enkil_cog }} is key here, as it allos {{ thucimia }} to bleed

Tips & Tricks

The key tactic of the deck is to use {{ enkil_cog }} to bleed during the prey's turn, thus ensuring they cannot use {{ archon_investigation }}. Then boost {{ thucimia }}'s bleed as much as possible, and use {{ deed_the_heart_s_desire }}, {{ direct_intervention }} and {{ touch_of_clarity }} to prevent {{ bounces }}.

{{ secure_haven }} is the only protection {{ thucimia }} can get from rushes, apart from sacrificing another vampire to the block.

Not only can {{ direct_intervention }} be used during an {{ enkil_cog }} bleed in another Methuselah's turn, it is also a safeguard to cancel any potential contesting {{ enkil_cog }} on the table.

{{ wash }} can be used to counter permanent intercept like {{ krcg_news_radio }}, as well as bloat Master cards like {{ villein }} and, maybe more importantly, any tentative to contest {{ secure_haven }}.
