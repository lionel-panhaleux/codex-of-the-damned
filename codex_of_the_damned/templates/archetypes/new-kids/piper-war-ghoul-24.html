Highlights

This is a rush deck using multiple {{ war_ghoul }} to rush the opponents vampires.

The idea is to get a {{ war_ghoul }} into play as soon as possible, using a low capacity Tzimisce, {{ piper }} to recruit it, and {{ jake_washington }} as a sacrifice. The {{ war_ghoul }} then proceeds to taking down every vampire on the table, using {{ fake_out }} and {{ trap }} as needed.

Other allies and retainers can be recruited to be sacrificed to more {{ war_ghouls }}. {{ ossian }} and {{ vozhd_of_sofia }} if drawn, may even be better than a {{ war_ghoul }}.

The deck uses low capacity vampires with {{ dom }} to bounce bleeds and {{ pro }} for unlock and intercept. {{ deep_ecology }} and {{ one_with_the_land }} allows them to go up to 4 intercept against direct actions. Pool damage is done with {{ dragonbound }} and naked bleed once all the prey's vampires are down.

Tips & Tricks

{{ asanbonsam_ghoul }} and {{ carlton_van_wyk }} can help with the defense, they can both get up to 2 intercept with {{ the_unmasking }}. and {{ direct_intervention }} is a ultimate recourse against ally theft cards like {{ entrancement }}

This a fast and aggressive deck and will likely attract the table attention. It should try to keep control of the whole table if possible.

Letting some vampires live on the predator side can be a good way to bounce their bleed and deliver additional pool damage to the prey.

The deck is particularily fragile against political decks, so voter should probably be taken out first.
