Highlights

This is a vote deck using Malkavian Justicar and Princes, together with the infamour {{ arishat }}.

The idea is to get use {{ govern_the_unaligned }} to influence more vampires fast and cheap as usual, then proceed to oust the prey with votes like {{ parity_shift }} and {{ kine_resources_contested }}.

The stealth is overwhelming, with {{ forgotten_labyrinth }} combined with {{ lost_in_crowds }} letting the vampires achieve 5 stealth on undirected actions without breaking a sweat. And if anyone manages to go that high, there is always {{ elder_impersonation }}.

Votes are secured by the vampires titles combined with {{ arishat }}'s ability and {{ condemnation_mute }}. {{ deflection }} is used for bleed defense, and {{ obedience }} to avoid fights altogether.

Tips & Tricks

The deck packs no wake nor unlock, so one must be cautious in keeping a vampire unlocked when needed, to play {{ deflection }} and {{ obedience }}.

{{ parity_shift }} is key in the deck: it aims to influence more vampires to get its pool under the prey's pool, so {{ parity_shift }} can be used.

{{ unleash_hell_s_fury }} is an excellent defense against rushes, especially high capacity rusher who would not be impaired by {{ obedience }}.

{{ juliet_parr }} is a quasi-star here, her ability helps to cycle unneeded cards easily.

{{ my_kin_against_the_world }} is a nice tool: either unlock vampires for defense, or use after voting to bleed with the remaining {{ govern_the_unaligned }} if the prey's defense is weak enough, for a brutal lunge.

Recursion is strong, alowing to fetch whatever is need, be it a vote, an {{ unleash_hell_s_fury }}, more {{ deflection }}, or more {{ stealth }}. {{ ashur_tablets }} and {{ the_sargon_fragment }} allow to fetch back whatever is most impactful for the table.
