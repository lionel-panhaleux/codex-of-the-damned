Highlights

This is a stealth & bleed starring {{ kalinda }} and relying on {{ haqim_s_law_retribution }} to increase its bleed, and {{ protected_district }} for defense.

The idea is to influence {{ kalinda }}, as her ability lets her bleed then unlock, to be ready to defend with {{ protected_district }}.

Stealth is pretty high, with Obfuscate {{ OBF }} modifiers doing their usual job. {{ swallowed_by_the_night }} and {{ resist_earths_grasp }} are favored because they can be used for stealth or discarded by {{ haqim_s_law_retribution }} for more bleed power.

The combat module is mostly defensive, with a lot of maneuvers and some combat ends. But it can still surprise an unwary opponent with {{ hunger_of_marduk }} and {{ pursuit }}.

Tips & Tricks

The deck packs a small political module that can help with the lunge, using {{ camarillas_iron_fist }}. It can be negotiated witht he table, or pass thanks to the static votes available or {{ alamut }}, if it is available. {{ banu_haqim_justicar }} should be easy to pass and also provide more votes.

The {{ ashur_tablets }} & {{ infernal_pursuit }} combination is an efficient way to fix and improve the cards flow if needed.

The deck is close enough to {{ haqim_royalty }} that it can keep the opponents guessing for a while. It makes them cautious about entering combat with the vampires, seeing how that archetype is so strong in combat. It is a lever one should not hesitate to exploit when playing this deck.

Variants

{{ alamut }} can be doubled, and {{ web_of_knives_recruit }} is a nice fit.
