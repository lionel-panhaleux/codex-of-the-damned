Highlights

This is a political deck using Toreador who have Fortitude {{ for }}, to allow for more actions with {{ forced_march }}. It uses the unique {{ toreador_grand_ball }} to avoid blocks on political actions.

Stealth is los, the deck relies on the {{ grand_ball }} to avoid dedicated blockers. That said, {{ creepshow_casino }}, {{ camarilla_conclave }} and {{ perfect_paragon }} can provide just the stealth needed to get an action through. Political actions provide pool damage and a bit of bloat, and Presence modifiers are used to ensure they pass without issue: {{ awe }}, {{ bewitching_oration }}, {{ perfect_paragon }} and {{ scalpel_tongue}}.

Using {{ voter_captivation }} and {{ villein }}, the deck ensures a solid bloat. But it also packs some bleed defense with {{ telepathic_misdirection }} and {{ protected_resources }}.

The combat module is strictly defensive, with just {{ majesty }} for combat ends.

The deck does not aim to defend the {{ grand_ball }}. With its solid bloat, it just replaces it when it is lost, and with {{ the_parthenon }} and {{ ashur_tablets }} can even end up playing and replacing two of them. Of course, it is still worth defending with {{ second_tradition }} and {{ majesty }} if it is ont too risky.

Tips & Tricks

Multiple {{ grand_balls }} can be stacked using the same "host", ie. locking only one vampire for all of them.

{{ diana_iadanza }} is a quasi-star in the deck. Her ability lets her avoid {{ delaying_tactics }}, and with superior Fortitude {{ for }}, she can use {{ forced_march }} for stealth. So in the mid-game, she might be the one less needing a {{ grand_ball }}, if the neighbours have low enough intercept.

{{ third_tradition_progeny }} allows to create a cheap host for the {{ grand_ball }}, freeing an older vampire.

Recursion is strong in this deck, and {{ the_sargon_fragment }} can allow a {{ grand_ball }} guest to fetch just the right card at the right time. Together with {{ ashur_tablets }}, it opens nice windows of opportunities in the end game.

The surprise {{ aire_of_elation }} can take an opponent by surprise and provide just the damage needed for a lunge, or just be discarded to keep the opponents guessing. It can always be brought back in hand with recursion if needed, particularily in the end game duel.

Variants

{{ closed_session }} is an option to consider, and {{ camarilla_segregation }} is often included. {{ charismatic_aura }} can be added against {{ immortal_grapple }}.

It is also possible to run this without {{ ashur_tablets }}, see {{ valdemir_santos_deck }}.
