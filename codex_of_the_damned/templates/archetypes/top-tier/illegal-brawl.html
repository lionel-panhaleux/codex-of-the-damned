Highlights

This is a toolbox deck using Brujah {{ brujah }} barons.

The deck has a strong combat module relying on the versatile "three-way" combat cards {{ diversion }}, {{ dust_up }}, and {{ bollix }}, as well as good old Portence {{ pot }} staples like {{ immortal_grapple }} and {{ disarm }}.

It uses versatile actions in {{ illegalism }} and {{ line_brawl }} to either bleed or rush, with {{ power_of_one }} as a stealth and enahncer modifier when bleeding.

On the defensive side, it relies on the defensive abilities of Anarch Barons: {{ organized_resistance }} for unlock and {{ bait_and_switch }} for bounces.

{{ villein }} and {{ blood_doll }} are used to bloat by leeching some blood of the vampires, while {{ carfax_abbey }}, {{ papillon }} and {{ taste_of_vitae }} are used for blood regeneration.

Tips & Tricks

{{ new_carthage }} is a momentus boost given the crypt, bumping the bleed and vote power of all the Barons. It explains why it is included twice.

{{ garibaldi_meucci_museum }} is incredibly versatile, fetching the required action, modifier or bounce, as well as being able to contest it to avoid its combat ends effect in other decks.

{{ powerbase_los_angeles }} is great here: both the unlock and additional discard are useful, and the deck has no shortage of Anarch cards to discard.

{{ open_war }} can help control the whole table in the mid or late game, providing enough rush capability to take down even the most resilient opponents.

{{ an_anarch_manifesto }} is a nice inclusion. The deck does not seek stealth that much, but it is free and the downside is almost negligible thanks to {{ dust_up }}.

Variants

The Resistance variant includes much more {{ organized_resistance }} for more reactions. It is not that much intercept, but it allows for the inclusion of good cards that require a defense, like {{ constant_revolution }} and {{ stolen_police_cruiser }}. See {{ tomáš_marek_s_deck }}. Furthermore, {{ the_anarch_free_press }}, {{ mr_winthrop }} can be included for more intercept.

The combat module can vary. From Darby's own admission, {{ thrown_gate }} is not ideal. {{ diversion }} is not always included, though most keep the {{ dust_up }} & {{ bollix }} combo. It can also forgo three-ways and rely on classic {{ POT }}{{ CEL }} cards like {{ pursuit }}, {{ preternatural_strength }}, and {{ roundhouse }}. Finally, some focus on {{ concealed_weapon }} and {{ magnum }}, like in {{ héctor_ordóñez_deck }}.

Not everyone uses {{ illegalism }}: {{ enchant_kindred }} as in {{ erik_albenze_s_deck }}, or {{ show_of_force }} as {{ cicero_jannyson }}, are alternatives.

The Chimerstry {{ chimerstry }} variant includes some {{ chimerstry_card }} to get stealth out of {{ illegalism }}, but also enable {{ dabbler }}. See {{ kelly_lyons_deck }} for example.

The Primogen variant uses a lot of {{ protected_district }}, and lower capacity vampires to swarm the prey a bit by focusing on the bleed part of things. See {{ norman_brown_s_deck }}
