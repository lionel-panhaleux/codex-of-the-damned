Highlights

This is a Ravnos {{ ravnos }} stealth & bleed deck using {{ break_the_bonds }} to bleed fast and hard.

It is a pretty straightforward deck. Influence a Ravnos {{ ravnos }}, bleed with {{ break_the_bonds }}, use {{ OBF }} for stealth and bring more Ravnos {{ ravnos }} in play. Rince and repeat.

The deck relies on {{ memory_rift }} as its main stealth device, because it does not jam the hand, but also its combat ends effect can be frustrating to blockers and help jam their hand on combat cards.

At some point, {{ club_illusion }}, {{ week_of_nightmares }} or both hit the table, and the bleeds become an unstoppable wave that sweeps the table.

In the meantime, simply use {{ visions_of_zapathasura }} to reduce the predator's bleed.

{{ dabbler }} is, as in {{ platinum_revelation }}, a very good archetype to have, because it can let the vampires unlock for defense after a bleed or, maybe more importantly, get some blood back to feed a {{ club_illusion }} while they cannot hunt thanks to {{ week_of_nightmares }}.

Tips & Tricks

Since the deck uses no {{ bounce }}, {{ narrow_minds }} is a natural inclusion. {{ spying_mission }} can also be used to avoid damaging an ally when bounced by the prey.

{{ field_training }} is a nice inclusion, allowing to provide superior Obfuscate {{ OBF }} to the few vampires who do not have it in the crypt.

{{ wash }} can be used to counter permanent intercept locations like {{ krcg_news_radio }}, but also bloat cards like {{ villein }}, thus amonting to a bleed in terms of pool loss.

Variants

{{ guard_dogs }} can be used for additional unlocks to enable more {{ visions_of_zapathasura }}, and {{ majesty }} can be added for a bit of combat defense. Given {{ break_the_bonds }} effect, {{ faceless_night }} is actually stronger than usual, so more copies can be nice.

{{ visions_of_zapathasura }} can be omitted altogether for a more streamlined version optimizing the bleed. More bleat is required then, like including a lot of {{ villein }} to recoup pool, see {{ darby_keeney_s_deck }} for example.
