Highlights

This is a stealth & bleed deck that uses {{ the_british_museum }} to unlock after fetching {{ living_manse }}, enabling a rapid ramp-up to high pool damage.

As usual, the highest capacity vampire should be influenced first, to use {{ govern_the_unaligned }} for cheaper and faster subsequent influences.

The deck then takes some time to set up, getting all sorts of equipments and retainers while defending against bleeds with {{ bait_and_switch }} and {{ deflection }}.

When ready, it then proceeds to bleed hard and remove its preys one after the other.

Tips & Tricks

{{ garibaldi_meucci_museum }} can be used to bring back {{ bait_and_switch }} or {{ eat_the_rich }}, depending on the need, but it can also be used to stop a threatening combat if the opponent is Anarch.

{{ heidelberg_castle_germany }} allows to move bleed, but also displace {{ incriminating_videotape }} or {{ the_signet_of_king_saul }} after a bleed, double-dipping from the equipments.

{{ nra_pac }} is a great adjunction if drawn, allowing for an unlock at the end of turn to be ready for defense, after having used {{ the_british_museum }} to unlock from an equipment action during the turn, then bleed. A nice tactical move known as "the Darby dance"

Given the deck's strategy with its well-defined setup and lunge steps, {{ heart_of_nizchetus }} is a great asset, allowing to filter the right cards for the moment, so it comes in 2 copies.

Variants

This deck was imagined by Darby Keeney, so his {{ latest_variation }} is worth a look: after all, he is the "Darby dance" inventor, so he knows what he is doing here. {{ dominate_kine }} is a nice inclusion, and equipments focus more on blood regeneration with {{ cooler }} and {{ palatial_estate }}. {{ powerbase_savannah }} is another interesting inclusion that can defend against rushes.
