Highlights

This is a toolbox deck using Banu Haqim {{ banu_haqim }} Princes and Justicar.

The key card is {{ haqims_law_retribution }}, providing strong bleeds from a very heavy combat module. As most royalty builds, defense takes the form of {{ second_tradition_domain }} for blocks.

The combat module is dreadful. {{ hunger_of_marduk }} coupled with additional strikes from {{ cel }}, can consistently empty the blood from opponents vampires, and the maneuverability should guarantee the range choice most of the time (and avoid {{ immortal_grapple }} or aggravated {{ claws_of_the_dead }}).

Tips & Tricks

The main weakness of the deck is its light defense. Intercept does not often go above the 2 from {{ second_tradition_domain }}, and this might not be enough to stop well-stealthed vote or bleed decks. The absence of {{ bounce }} makes bloat necessary: here {{ villein }}, {{ anathema }} and the excellent {{ priority_contract }}. Although unique, it will not stay in play for long if the target is chosen well given the heavy combat module, so it's an almost guaranteed 3 pool. It also works if the prey gets ousted (since their vampires leave the game), and it's a trifle!

{{ resist_earths_grasp }} and {{ swallowed_by_the_night }} are versatile here: aside from combat, they can provide stealth and, in case the prey does not even try to block, they can be discarded to {{ haqims_law_retribution }} for additional bleed.

{{ hunters_mark }} provide rushes that can be used anywhere for table control, but mostly as a countermeasure to slow down a strong predator, since {{ priority_contract }} cannot be used backwards.

The {{ ashur_tablets }} & {{ infernal_pursuit }} combination is not a constant, and {{ haqims_law_retribution }} is often included in higher numbers.

Variants

The combat module varies depending on the expected meta. {{ hunger_of_marduk }} can be run in much more copies, {{ flash }}, {{ blood_rage }}, {{ soul_burn }}, {{ walk_of_flame }}, {{ lightning_reflexes }} or {{ fast_hands }} can be included. Another interesting option, although a bit expensive, is to use some {{ biothaumaturgic_experiment }}, like in {{ luis_otavio_franco_martins_deck }}.

There are a few options to consider: {{ the_khabar_community }} is a common inclusion used to lunge, especially in decks including more {{ swallowed_by_the_night }}. On the defense side, {{ market_square }}, {{ mr_winthrop }}, {{ sport_bike }} and even {{ bowl_of_convergence }} can be included. On the actions side, the deck can include some {{ judgment_camarilla_segregation }} or {{ khabar_glory }}, and in terms of events, {{ narrow_minds }} and {{ scourge_of_the_enochians }} are also common.

The deck can be a bit more political and include {{ alamut }} or even {{ the_black_throne }} for more votes, see for example {{ darby_keeney_s_deck }}. {{ nsa_trio }} is a nice addition against {{ sce }}.

On the other end, {{ kasim_bayar }} can be excluded for a swarmier using {{ the_embrace }}, {{ web_of_knives_recruit }}, or {{ third_tradition_progeny }}, like in {{ pauli_tuomiluoma_s_version }} or {{ juan_luis_mejías_luna_s }}.

Finally, it's worth mentioning there are other approaches to the clan, leading to different archetypes, like {{ banu_politics }} or {{ kalinda }}.
