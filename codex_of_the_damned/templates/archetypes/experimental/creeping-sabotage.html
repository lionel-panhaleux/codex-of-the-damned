Overview

This deck uses neither bleed nor vote to deliver pool damage but uses {{ anarch_revolt }}, {{ army_of_rats }} and {{ creeping_sabotage }}. This way of ousting one's prey disables a lot of usual defensive modules such as {{ bounce }} or {{ delay }}.

This strategy costs a bit of blood and those cards need to be defended. We go mid-cap for speed and {{ a_b }}. The key here is having the upper hand in {{ combat }} hence the use of {{ POT }} as a core discipline of the deck as it is great at dealing high damage.

Nosferatus {{ img_clan_nosferatu }} are good at {{ ani }}, {{ obf }} and {{ pot }} disciplines, {{ slag }} and {{ ryan }} for example. Some even have a built-in {{ rush }} action ({{ ebanezer_roush }} and {{ gustave_brunnelle }}) but they are not used because of their higher capacity and the low presence of mid-cap {{ POT }} vampires in their grouping.

This {{ rush }} strategy leads to {{ finnish_politics }}. Its point is to use {{ rush }} as a means of negociation with the other players. Played with {{ immortal_grapple }}, it becomes a huge threat to {{ s_ce }} defensive modules. It is also the only defense against an aggressive predator.

We go with {{ bums_rush}} and {{ haven_uncovered }} despite {{ ANI }} giving access to {{ deep_song }}. As the point is to disable {{ bounce }}, it would go against that objective because {{ bleed }} for 2 are nice to bounce.

Range Management

{{ range_management }} is sometimes hard to deal with. Here are some {{ POT }} cards that can be used and each one of them is a metacall answer:

{{ thrown_sewer_lid }} is great against {{ CEL }} guns decks because of the relatively high density of maneuvers in the deck. {{ anson_guns }} is a common build for this kind of deck.

{{ slam }} works fine against decks with few maneuver cards but yet depend on it. For example, {{ ANI }} decks like {{ nananimalism }} relies heavily on {{ aid_from_bats }} but it does not play other cards to maneuver.

{{ improvised_tactics }} is a more neutral card. It is possible to cycle it away at no cost and even get an extra card from time to time with a vampire with {{ aus }}. This card may be played when few long range decks are expected.

This deck has some other ways to deal with {{ range_management }} as it may be a weakness of {{ POT }} oriented combat modules:

It is possible to use {{ stealth }} to avoid blocks on {{ bums_rush }} to profit from its built-in optional maneuver.

{{ shadow_boxing }} and {{ swallowed_by_the_night }} provide maneuver options. The other parts of those cards are also playable and provide versatility to the deck.

Tournament Version

This deck has been inspired by two tournament archetypes: {{ nananimalism }} and {{ euro_brujah }}. Despite their different core discipline {{ ANI }} and {{ DOM }}, the density of the {{ modules }} were of great help to design this deck.

It can be possible to choose {{ ANI }} as the combat discipline using {{ aid_from_bats }} and {{ carrion_crows }} to deal damage. The integration of {{ creeping_sabotage }} in {{ nananimalism }}'s game plan may be a development axis.

It can also be possible to play a wall version of this deck using {{ AUS }} and/or {{ ANI }} to {{ intercept }} the directed actions against the {{ creeping_sabotage }}.
