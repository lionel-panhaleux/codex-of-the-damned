Highlights

This is a rush deck based on mid-cap {{ POT }} vampires.

Ordinary rush deck uses {{ ANI }} as the key discipline because {{ deep_song }} is both the means to rush and the means to oust the prey. Other {{ combat }} decks can enter combat using vampires' special action, like {{ beast_leatherface }} for example.

There is no alternative to {{ deep_song }} for {{ POT }} decks and there are only two Nosferatu {{ img_clan_nosferatu }} able to enter combat as a special action: {{ ebanezer_roush }} and {{ gustave_brunnelle }}.

This means that the deck uses {{ bums_rush }} as a means to rush and {{ creeping_sabotage }} as a means to oust the prey.

Tips & Tricks

As {{ creeping_sabotage }} only deals damage to the prey, it is less likely that other players will try to use actions to destroy it. {{ bums_rush }} can then be used to reduce the number of {{ actions }} that the prey can use to attempt to destroy it.

If a prey or a predator has a good {{ range_management }} capability, {{ stealth }} can be used to secure the optional maneuver from {{ bums_rush }}.

Using {{ the_warrens }} early in the game can dissuade some Methuselah to use actions to attempt to destroy {{ creeping_sabotage }}.

There are some flex slots in the combat module that deals with ranged combat: {{ thrown_sewer_lid }}, {{ slam }}, {{ earthshock }}, {{ improvised_tactics }}. Choosing one over the others depends on what type of ranged fights you are expecting.
