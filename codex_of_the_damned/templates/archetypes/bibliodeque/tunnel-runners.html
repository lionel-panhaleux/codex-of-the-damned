Highlights

This is a stealth & bleed deck using {{ tunnel_runner }}. An Akunanse {{ akunanse }} is required to get the weremoles out, and {{ jacob_fermor }} is expected to come out and boost their bleed power. {{ computer_hacking }} is used to help them bleed harder, too.

The deck relies on additional vampires with {{ dom }} for {{ deflection }} bounces. As the deck does not block so much, this is essential for defence.

Tips & Tricks

As the deck is supposed to get an Akunanse, {{ jacob_fermor }} then a low capacity vampire with {{ dom }}, it has a bit of each in the crypt and plays a bunch of {{ wider_view }} and an {{ anarch_convert }} to rotate through them if necessary.

{{ tunnel_runner }} are expensive allies, getting {{ powerbase_tshwane }} and/or {{ charisma }} out fast is a lot of help.

{{ predator_s_transformation }} provides additional stealth to get the allies into play without being blocked.

{{ aid_from_bats }} is used as a defensive combat card. {{ tunnel_runner }} can use it, but most Akunanse also can. The allies have {{ the_unmasking }} and the Akunanse their {{ predator_s_communion }} for some tactical intercept — vampires that do not fight aggressively, as they have very few combat capabilities themselves.

Variants

There is no variant to this deck. The decklist given in example won the 2013 North American Championship (NAC).
