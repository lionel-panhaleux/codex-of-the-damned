Highlights

This is a weenie bleed deck using the {{ dom }} discipline. The idea is to get as many low capacity vampires out as fast as possible, and bleed the preys one after the other as fast as possible.

{{ govern_the_unaligned }} combined with {{ conditioning }}, {{ seduction }}, and {{ command_of_the_beast }} make for hard to block dreadful bleeds for 7.

As the game progresses, some {{ dom }} bounces can be used to add to the damages if the predator becomes aggressive.

The Master module uses {{ dominate }} to enhance the vampires, but also {{ the_parthenon }} and {{ ashur_tablets }} for a bit of recursion. {{ jake_washington }}, {{ life_in_the_city }} and {{ giant_s_blood }} are used to bring back some blood on the vampires.

The combat modules uses only {{ absorb_the_mind }} for defence. It is a bit weak, but {{ dom }} has nothing better to offer.

Tips & Tricks

Well-timed {{ misdirection }} and {{ pentex_subversion }} can help for the lunge.

Variants

The Agent of Power variant cuts the {{ ashur_tablets }} and uses {{ agent_of_power }} to let more vampires use superior {{ DOM }}, like in {{ manoel_pinheiro_s_deck }} from 2014.

The Aim & Chain variant uses {{ target_vitals }} and {{ zip_gun }} with a bit of {{ for }} prevention for the combat module, as in {{ vincent_ripoll_s_deck }} from the 2009 EC Day 1.
