Highlights

This is a toolbox deck centered around the {{ CEL }}{{ AUS }}{{ QUI }} disciplines, with {{ dmitri_borodin }} as a star.

{{ truth_of_blood }} and {{ trochomancy }} are used to bleed. {{ disguised_weapon }} along {{ magnum }} and {{ sniper_rifle }} for combat. {{ cel }} provides some maneuverability with {{ pursuit }}.

{{ AUS }} provides intercept and bounces, along the excellent {{ black_sunrise }} for additional unlocks. There is a bit of permanent intercept and {{ smiling_jack_the_anarch }} is supposed to be kept on the table to increase pool damage.

Tips & Tricks

{{ enkil_cog }} is an excellent bleed enhancer, but an {{ archon_investigation }} on the prey side is dangerous if it is used in conjunction with {{ trochomancy }}. Using {{ enkil_cog }} ability to bleed during the prey's turn prevents the use of {{ archon_investigation }} (if the bleed is not bounced): as it is an out-of-turn master, the prey can not play it during her own turn.

{{ the_fourth_cycle }} is included as a solution against problematic Gehenna card but since {{ anthelios_the_red_star }} ban in 2016 it should not be included in the deck anymore. {{ the_unmasking }} or a {{ dragonbound }} are not problematic and {{ veil_of_darkness }} is too rarely played to be considered a threat.

Variants

There is no major variant for this archetype. The crypt can be altered a bit, {{ infernal_pursuit }} added to help with the rotation and {{ target_vitals }} for more damages. Instead or along the {{ trochomancy }}, other permanent bleed enhancers can also be added, like {{ js_simmons_esq }} and {{ tasha_morgan }}.
