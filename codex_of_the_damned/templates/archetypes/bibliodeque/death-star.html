Highlights

This is a breed & boon anarch political deck using {{ alamut }} and {{ fee_stake }} actions to get votes, plus {{ the_embrace }} and {{ web_of_knives_recruit }} for breeding, {{ consanguineous_boon }} and {{ exclusion_principle }} for bloating.

The idea is to use {{ revolutionary_council }} with a lot of anarchs at hand to deliver more than a dozen pool damage in a single blow, hence the "Death Star" name.

The deck has a strong stealth module with {{ OBF }}.

Tips & Tricks

During the beginning of the game, the deck should focus on breeding and bloating, surviving its predator.

{{ crimeth_inc }} and {{ firebrand }} are used to improve {{ a_b }}.

{{ alamut }} should be resplenished as much as possible. Vampires with {{ obf }} are supposed to get some bleed through (even for 1 pool) to get tokens on it fast. If drawn, {{ club_illusion }} make these bleed stronger.

In late game, {{ garibaldi_meucci_museum }} helps get some {{ revolutionary_council }} back in hand to finish the table.

Variants

This decklist is quite famous, as Matt Morgan won the North American Championship (NAC) first day with it, only to see Ben Peal win the final on day 2 with an almost identical decklist.

{{ karl_schaefers_deck }} from 2019 is a variant using Daughters of Cacophony {{ daughter_of_cacophony }} instead of Assamite. {{ alamut }}, {{ obf }} stealth modifiers and {{ web_of_knives_recruit }} are lost, but the Daughters  {{ daughter_of_cacophony }} have access to {{ pre }} vote modifiers and combat ends, {{ conductor }}, {{ the_missing_voice }}, {{ lily_prelude }} and {{ madrigal }}, making up for it.
