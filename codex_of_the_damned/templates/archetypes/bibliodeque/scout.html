Highlights

This is a Vote deck using {{ scout_youngwood }} for her {{ MEL }}{{ OBF }}{{ PRE }}{{ for }} disciplines and unique access to {{ shattering_crescendo }}, {{ lily_prelude }} and {{ reckless_agitation }}.

{{ villein }} and {{ zillah_s_valley }} are used to get more vampires out fast and cheap.

The deck has serious stealth, with most vampires having {{ OBF }}, and plays {{ heart_of_the_city }} to enhance the bleed in the late game.

As all vampires except {{ scout_youngwood }} are Princes and Justicars, {{ parity_shift }} is heavily used both for pool damages and bloat.

Tips & Tricks

{{ dmitra_ilyanova }}'s ability is a great way to get more blood on the table for those {{ villein }} and {{ reckless_agitation }}.

With only 5 votes amoung its two star vampires, the deck can struggle to get the vote lock against other political constructs. Hence the inclusion of {{ legendary_vampire }}, {{ bewitching_oration }} and {{ perfect_paragon }} to get critical votes through. If the pool allows it and she is drawn, getting {{ arishat }} out is a good way to handle an Inner Circle, too.

Combat is purely defensive and uses only {{ majesty }}.

Variants

The archetype is seldom played and not many variations exist. Using a different crypt around can allow for different plays, for example using {{ unmada }} and {{ benjamin_rose }} for {{ AUS }} and the excellent {{ telepathic_misdirection }} bounce. {{ djordje_bjelajac_s_deck }} from the 2011 EC FCQ is a good example of that.
