Highlights

This is a stealth vote & bleed deck using Multi Master Phase Actions (MMPA). It uses {{ huitzilopochtli }}, {{ the_parthenon }} and {{ rumors_of_gehenna }} to get more Master actions, then {{ villein }} and {{ information_highway }} to get big capacity vampires into play fast and cheap.

{{ ashur_tablets }} and {{ liquidation }} are used to bloat and get some recursion to adapt the deck content to the table context for maximum efficiency.

The deck delivers pool damage from stealthed naked bleeds from the Inner Circle {{ arika }} and {{ leandro }}, and also damage-dealing votes.

{{ majesty }} is used for combat protection, {{ deflection }} to bounce bleeds and {{ OBF }} for high stealth using {{ into_thin_air }} and {{ elder_impersonation }}.

Tips & Tricks

If drawn, getting {{ huitzilopochtli }} into play first for more Master actions is best.

This is an old archetype, and this version uses many cards since banned: {{ anthelios_the_red_star }}, {{ temptation_of_greater_power }} and {{ liliths_blessing }}. No doubt removing them makes the deck less efficient, and modifications would have to be made.

Variants

No variant of this deck exist per se, although one could argue that the {{ girls_will_find_inner_circle }} archetype is a more or less direct heir to this archetype.

Many minor variations on the crypt or minion cards exist. Including {{ etrius }}, {{ obedience }} for combat defence, {{ zillahs_valley }} for crypt acceleration, some {{ dom }} bleed enhancers like {{ conditioning }} or {{ carlton_van_wyk }} for defence, are quite usual. {{ ruben_feldmans_deck }} from 2009 is a good example.
