Highlights

This is a toolbox deck starring {{ card("Anson") }} alongside {{ card("Alexandra") }} and {{ card("Anneke") }}, hence the "AAA" name.

The deck is well-balanced, using {{ card("Second Tradition: Domain") }} for wake and intercept, {{ card("Concealed Weapon") }} + {{ card("44 Magnum") }} with C for combat, {{ card("Parity Shift") }} and {{ card("Aire of Elation") }} for pool damage and {{ card("Anson") }}'s ability to bloat and use recursion with {{ card("Ashur Tablets") }} and {{ card("Liquidation") }}.

{{ card("Resist Earth's Grasp") }} can provide a bit of stealth or be used in combat if required.

{{ card("Heart of the City") }} and {{ card("Codex of the Edenic Groundskeepers") }} are good bleed enhancers for the late game.

Tips & Tricks

{{ card("Entrancement") }} can help get rid of cumbersome allies, {{ card("Scourge of the Enochians") }} is there to deal with breed constructs.

As the game progresses, {{ card("Ashur Tablets") }} are used for recursion to adapt the deck content to the table context, bringing back more of what is really needed in the deck.

Variants

The Anson Guns variant focuses on {{ card("Anson") }} and drops its Toreador companions to get cheaper vampires with C, like in {{ link("/deck-search", _("Tomasz Izydorczyk's deck"), twda_id="2019pncwp") }} from 2019.

The Grand Ball variant ditches the combat module and uses {{ card("Alexandra") }}'s ability with {{ card("Toreador Grand Ball") }} to provide unblockable votes, like in {{ link("/deck-search", _("Martin Miller's deck"), twda_id="2013cncpf2") }} from 2013.

The Aching Beauty variant also replaces the combat module with defensive {{ card("Majesty") }}, but uses {{ card("Aching Beauty") }} in numbers to make the vampires actions hard to block, like in {{ link("/deck-search", _("László Walter's deck"), twda_id="2012ecsod") }} from 2012.

The AAAAA variant keeps {{ card("Alexandra") }} and {{ card("Anneke") }} and uses the {{ card("Alastor") }} + {{ card("Assault Rifle") }} combination (hence the five "A"), like in {{ link("/deck-search", _("Izaak Havelaar's deck"), twda_id="2010kaldecq") }} from 2010.
